# getWhois

WebExtension to Get Whois information.

## Install

`pnpm i`

## Development

### Firefox

`pnpm dev`

### Chrome

`pnpm dev:chrome`

## Production

### Firefox

`pnpm build`

### Chrome

`pnpm build:chrome`

## Archive

### Firefox

`pnpm zip`

### Chrome

`pnpm zip:chrome`
